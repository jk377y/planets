## JSON-Server [https://github.com/typicode/json-server](https://github.com/typicode/json-server)
Use local JSON data to create a REST API<br>
```db.json```<br>
Locally installed my-json-server<br>
```npm install json-server```<br>
Start the server<br>
```npx json-server db.json```<br>
This starts the server at http://localhost:3000<br>
